Berikut adalah tugas game tebak angka sederhana yang saya buat menggunakan Python. Program ini dibuat untuk memenuhi tugas pemrograman berorientasi object.

## Deskripsi Singkat

Game ini adalah program tebak angka di mana komputer akan memilih satu angka rahasia antara 1 sampai 10 secara acak. Pemain harus menebak angka tersebut dengan clue yang diberikan setiap tebakan salah.

## Fitur yang Ada

- **Random angka 1-10**: Komputer memilih angka secara otomatis
- **Petunjuk tebakan**: Memberitahu apakah tebakan terlalu kecil atau terlalu besar
- **Hitung percobaan**: Menghitung berapa kali pemain menebak sampai benar
- **Validasi input**: Memastikan input hanya angka 1-10
- **Keluar kapan saja**: Ketik 'exit' untuk berhenti
- **Main lagi**: Bisa restart game tanpa menutup program
