# 🔢 Array Sorting Program

Program sorting array sederhana dengan berbagai algoritma sorting (Bubble, Selection, Insertion, Quick, Merge) dan mode input interaktif. Dibuat untuk tugas Struktur Data.

---

## ✨ Fitur

- ✅ **5 Algoritma Sorting**: Bubble, Selection, Insertion, Quick, Merge + Python Built-in
- ✅ **2 Mode Pengurutan**: Ascending (kecil→besar) & Descending (besar→kecil)
- ✅ **Input Interaktif**: User input angka sendiri via keyboard
- ✅ **Validasi Input**: Anti-crash jika input salah
- ✅ **Support Desimal**: Bisa mengurutkan angka bulat & desimal
- ✅ **Data Aman**: Array asli tidak berubah (menggunakan copy)

---

## 🚀 Cara Menjalankan

### Persyaratan
- Python 3.x

### Langkah-langkah

1. **Clone atau download** file `sorting.py`

2. **Jalankan program**:
   ```bash
   python sorting.py
